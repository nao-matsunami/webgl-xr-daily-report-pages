Depth Gate Tunnel

奥行き方向に並ぶゲートを順に通過していく最小サンプルです。 ポインタで消失点をずらし、各層が視差と色収差を伴って流れる構成にしています。

テーマ: depth staging / gate tunnel / ambient motion
入力: ポインタで消失点と視差を変更
変化: gate 間の揺れと速度差
狙い: UI ではなく空間遷移として情報の節を見せる
現在: focus drift は標準
次に足すなら、各ゲート面に HTML texture やラベル断片を貼り、注目時だけ一面が読めるようにすると 空間演出と情報導線がつながります。
