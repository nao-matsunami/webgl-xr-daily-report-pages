表現・インタラクションのアイデアExpression and interaction ideas

7/13 の semantic mask は「どの splat を意味的に残すか」でした。今日はそこに時間差を足し、cyan を live semantic、amber を前フレーム/遅延フレームとの境界、magenta を stale residue として扱います。鑑賞表現としては、言葉で選ばれた領域が生きていて、その周囲に過去の解釈が残像として剥がれていく演出になります。制作診断としては、streaming、picking、semantic confidence、時間方向の re-association が崩れている場所を一目で見る overlay になります。

The 2026-07-13 semantic mask asked which splats should be kept semantically. Today adds temporal delay: cyan is live semantics, amber is the boundary against a previous or delayed frame, and magenta is stale residue. As a viewing experience, a language-selected region feels alive while older interpretations peel away as afterimages. As a production diagnostic, the same overlay reveals where streaming, picking, semantic confidence, or temporal re-association is breaking.

制作に使えそうな技術メモProduction notes

実装は依存なしの HTML + WebGL + GLSL。fragment shader 内で現在の semantic field と遅延 field を fbm で作り、その差分を residue として抽出します。複数の疑似 Gaussian splat は age、semantic fit、ポインタ距離を持ち、押下中は stale/risk を強調します。実 asset に進める時は、RGBA mask texture の R を semantic keep、G を live confidence、B を latency/residue、A を stream validity に割り当てると、SuperSplat selection、PlayCanvas SOG event、Babylon.js glTF splat 読み込み、three.js HTMLTexture 注釈を同じ規約で扱えます。

The implementation is dependency-free HTML + WebGL + GLSL. The fragment shader creates a current semantic field and a delayed field with fbm, then extracts their difference as residue. Each pseudo Gaussian splat has age, semantic fit, and pointer distance, and holding the pointer amplifies stale/risk signals. For real assets, assign RGBA mask texture channels as R = semantic keep, G = live confidence, B = latency/residue, and A = stream validity. That convention can unify SuperSplat selection, PlayCanvas SOG events, Babylon.js glTF splat loading, and three.js HTMLTexture annotations.
