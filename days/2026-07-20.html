表現・インタラクションのアイデアExpression and interaction ideas

中心案は『Temporal Hysteresis State Scope』です。ポインタの現在位置を current、ゆっくり追従する位置を history とし、その差分を lag / stale / retry の可視化に使います。押下中は retry pressure が増え、Space で synthetic upload spike が入り、queue が増えるほど画面粒度が粗くなります。短期では 1 枚の fragment shader で十分です。中長期では、PlayCanvas / SuperSplat の SOG chunk event、Babylon の retry cooldown、three.js の texture prewarm、Web Audio analyser を同じ timeline に入れ、操作後に残る時間差を作品の呼吸として設計します。

The core idea is Temporal Hysteresis State Scope. Pointer position is current, a slowly following position is history, and their difference drives lag / stale / retry visualization. Holding increases retry pressure, Space injects a synthetic upload spike, and higher queue values coarsen screen grain. Short term, one fragment shader is enough. Mid/long term, PlayCanvas / SuperSplat SOG chunk events, Babylon retry cooldown, three.js texture prewarming, and a Web Audio analyser should share one timeline so post-interaction delay becomes the experience's breathing.

制作に使えそうな技術メモProduction notes

状態 texture の設計は昨日の RGBA 案を継続します。R=stale hitch residue、G=warmed / uploaded quality、B=retry cooldown、A=validity とし、別 uniform で current pointer、history pointer、queue length、audio impulse、quality tier を持ちます。実装では history を shader 内で積分するより、JS 側で指数減衰として管理して uniform に渡すほうが WebGL1/2 で最小に保てます。requestIdleCallback は本物の GPU upload を直接保証しませんが、prewarm や軽い texture 生成のキュー長を表示するには十分です。次に実運用へ寄せるなら、renderer.initTexture() 相当のイベント、SOG lod-meta / chunk fetch、audio analyser の low/mid/high band を同じ state bus に流します。

Continue yesterday's RGBA state texture idea: R = stale hitch residue, G = warmed / uploaded quality, B = retry cooldown, A = validity, with separate uniforms for current pointer, history pointer, queue length, audio impulse, and quality tier. For implementation, keep history integration in JS as exponential decay and pass it as uniforms; this stays minimal across WebGL1/2. requestIdleCallback does not guarantee actual GPU upload timing, but it is useful for exposing prewarm or lightweight texture-generation queue length. To move toward production, feed renderer.initTexture-style events, SOG lod-meta / chunk fetches, and low/mid/high Web Audio analyser bands into one state bus.
