表現・インタラクションのアイデアExpression and interaction ideas

今日の中心案は「selection boolean を見える粒子場にする」ことです。cyan は add keep、red は subtract cut、violet は intersect residue、yellow は depth text gate。ポインタを動かすと選択 lens が移動し、押下中は intersect が強まります。作品表現としては、空間を選ぶ操作がそのまま色の層と残像になる。制作ツールとしては、semantic query、stream risk、LOD budget、MSDF label depth の衝突を一画面で確認できます。短期では単体 shader の体験、次の段階では SuperSplat selection を mask texture に焼き、PlayCanvas の SOG unload/reload event を alpha channel に重ねるのが良いです。

The core idea is to make selection booleans visible as a particle field. Cyan is add keep, red is subtract cut, violet is intersect residue, and yellow is a depth text gate. Moving the pointer moves the selection lens; holding/touching emphasizes intersection. As artwork, the act of selecting space becomes layered color and afterimage. As a production tool, it exposes collisions between semantic queries, stream risk, LOD budgets, and MSDF label depth in one view. Short term, this is a standalone shader interaction; next, bake SuperSplat selections into a mask texture and place PlayCanvas SOG unload/reload events in the alpha channel.

制作に使えそうな技術メモProduction notes

試作は依存なしの HTML + WebGL + GLSL。fragment shader 内で semantic field、stale field、subtract band、intersect seed、pseudo Gaussian splat を生成し、add / subtract / intersect を色分けしています。MSDF depth 書き込みの実 API は使わず、SDF 風の小さな glyph bars を depth gate として描き、文字情報が splat/residue の前後関係に入る見え方だけを先に検証しました。実装規約としては、R=selection keep、G=semantic confidence、B=latency/residue、A=stream validity を継続。今日からは別 texture または packed bits で operation mode(add/subtract/intersect)を持つと、SuperSplat の選択編集と shader overlay を同期しやすいです。

The prototype is dependency-free HTML + WebGL + GLSL. The fragment shader generates a semantic field, stale field, subtract band, intersect seed, and pseudo Gaussian splats, then color-codes add / subtract / intersect. It does not use the real MSDF depth-write API; instead, it draws small SDF-like glyph bars as a depth gate to test how text-like information can sit in front of or behind splat residue. Keep the convention R = selection keep, G = semantic confidence, B = latency/residue, A = stream validity. Starting today, add an operation mode texture or packed bits for add/subtract/intersect so SuperSplat selection editing can synchronize with the shader overlay.
